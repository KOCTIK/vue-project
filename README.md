# vue-project

## Application for managing space tourist flights

### Functionalities

1. Tourist management
   * List of tourists
   * Adding a tourist
   * Removing a tourist
   * Editiing a tourist          
     * Adding a flight (selection from existing flights)
     * Removing the flight
                    
2. Flight management
   * List of flights
   * Adding a flight
   * Removing a flight
   * Editing a flight          
     * Adding a tourist to the flight (selection of existing tourists)
     * Removal of a tourist from the flight

### Data structure

1. Tourist
   * First name
   * Last name
   * Gender
   * Country
   * Remarks
   * Date of birth
   * List of flights
          
2. Flight
   * Departure date and time
   * Arrival date and time
   * Number of seats
   * List of tourists
   * Ticket price
